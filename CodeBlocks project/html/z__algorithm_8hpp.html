Since the main point of the algorithm is to keep an invariant-bound [left;right], this method extends the bound to the right. Accordingly total_length is the length of the z array, haystack is the text we are searching in, needle is the string we are searching for, haystack_length is the length of the haystack, needle_length is the length of the needle, start is the the offset index we start the search from. The start is needed since the z array is dependent on it.
