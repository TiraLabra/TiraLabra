read_data Module¶

Luetaan laskentaa liittyvää dataa ja palautetaan data

read_data.read_data(filename='alkuarvot.txt')[source]¶

Yritetään lukea alkuarvot filestä alkuarvot.txt. Luetaan laskentaparametreja ja gridin tietoja. muutetaan siten että x->y ja y->x eli input tiedossa y tulee ensin koska se on matriisin 1. dimensio gridin koon määrittelyssä ja ytimien paikkojen määrittelyssä
