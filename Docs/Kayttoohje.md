## Käyttöohje

1. Tee karttatiedosto tekstieditorilla käyttäen seuraavia merkkejä:
aloituspiste A 
lopetuspiste B 
este # 
kulkemiskelpoinen maasto 0-9, missä 0 on helpointa ja 9 vaikeinta

Kartan tulee mieluusti olla suorakulmio (toimii muutenkin, mutta
tulostuksesta tulee hassu).

2. Aja jar-tiedosto komentokehoitteessa seuraavalla komennolla:
java -jar ohjelma.jar karttatiedosto.txt
