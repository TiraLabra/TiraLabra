Aihe: Tehdään ohjelma jolla voidaan kompressoida/pakata annettu data pienempään tilaan niin että se on vielä palautettavissa "bitilleen" ennalleen. Kyseessä siis häviötön kompressio.

Toteutus: Pakkaamiseen käytetään LZ77 algoritmia tai jotain siihen perustuvaa variaatiota esim. LZSS.

Kieli: Java
